🧩 Programa 13 – Suma de elementos de una lista
📌 Descripción

Este programa define una lista de números enteros y calcula la suma total de sus elementos utilizando un bucle for y una variable acumuladora.

Es una práctica orientada al procesamiento básico de estructuras de datos.

🧠 Conceptos trabajados

Listas predefinidas

Recorrido directo con for

Variable acumuladora

Operador +=

Procesamiento secuencial de datos

💻 Código
lista = [1, 2, 3, 4, 5]
resultado = 0

for elementos in lista:
    resultado += elementos

print(resultado)
