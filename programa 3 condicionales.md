🧩 Programa 2 – Condición if / else
📌 Descripción

Este programa solicita al usuario que ingrese su edad y determina si puede ingresar o no según sea mayor o menor de 18 años.

Es una práctica enfocada en el uso de estructuras condicionales para la toma de decisiones.

🧠 Conceptos trabajados

Entrada de datos con input()

Conversión de tipos con int()

Estructura condicional if / else

Operadores relacionales (<)

Lógica de decisión

💻 Código
edad = int(input("ingrese edad"))

if edad < 18: 
    print("es menor no puede pasar")
else:
    print("es mayor ya puede ingresar")
