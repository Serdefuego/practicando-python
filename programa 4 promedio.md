🧩 Programa 3 – Cálculo de Promedio
📌 Descripción

Este programa calcula el promedio de tres notas almacenadas en variables y muestra el resultado en consola.

Es una práctica orientada a comprender cómo combinar múltiples valores numéricos y realizar operaciones matemáticas básicas.

🧠 Conceptos trabajados

Declaración de múltiples variables

Operaciones aritméticas combinadas

Precedencia de operadores

División en Python

Impresión de múltiples valores en print()

💻 Código
nota_1 = 10
nota_2 = 7
nota_3 = 8

promedio = (nota_1 + nota_2 + nota_3) / 3

print("el promedio es de:", promedio)
