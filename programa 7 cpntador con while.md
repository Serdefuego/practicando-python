🧩 Programa 6 – Contador con while
📌 Descripción

Este programa imprime en pantalla los números desde 0 hasta 100 utilizando un bucle while.

El ciclo se ejecuta mientras la variable num sea diferente de 101, incrementando su valor en cada iteración.

🧠 Conceptos trabajados

Bucle while

Variable contador

Operador de incremento (+=)

Condición de corte

Control de iteraciones

💻 Código
num = 0

while num != 101:
    print(num)
    num += 1
